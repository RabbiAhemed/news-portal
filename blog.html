i) Difference between var , let & const

var variables can be updated and re-declared within its scope; let variables can be updated but not re-declared; const variables can neither be updated nor re-declared. They are all hoisted to the top of their scope. But while var variables are initialized with undefined , let and const variables are not initialized.

ii) Difference between arrow function & regular function

Regular function has this keyword but arrow function doesn't have it.

Arguments objects are not available in arrow functions, but are available in regular functions.

Regular functions created using function declarations or expressions are ‘constructible’ and ‘callable’. As regular functions are constructible, they can be called using the ‘new’ keyword. But, the arrow functions are only ‘callable’ and not constructible.

iii) Difference between map , forEach , filter & find

map : map is used for arrays.map executes the same code on every element in an array and returns a new array with the updated elements.

forEach : forEach is used to execute the same code on every element in an array but does not change the array and it returns undefined.

filter : filter checks every element in an array to see if it meets a certain criteria and returns a new array with all the elements that return truthy / matches for the criteria.

find : find also checks every element in an array to see if it meets a certain criteria and returns just the first element that return truthy / matches for the criteria.

iv) Why do we need/use template string

In-order to create a multiline string an escape sequence \n was used to give new line character. However, Template Literals there is no need to add \n string ends only when it gets backtick `` character.

To dynamically add values into new Template Literals expressions are used. The ${} syntax allows an expression in it that produces the value. This value can be a string stored in a variable or a computation operation.
